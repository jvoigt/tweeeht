## Description

This Project is based on the [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

It also used Mongo.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Running with a Database

You can connect to any running MongoDB.
There is a docker-compose.yml for convience.

````bash
# unit tests
$ npm run test

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
````
